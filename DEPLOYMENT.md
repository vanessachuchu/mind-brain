## 🚀 使用方式

### 本地開發
```bash
# 安裝依賴
npm install --legacy-peer-deps

# 啟動開發服務器
npm run dev

# 訪問 http://localhost:8080
```

### 部署到生產環境
```bash
# 構建生產版本
npm run build

# 預覽構建結果
npm run preview
```

## 🔧 配置說明

### AI API 配置
API 配置位於 `src/config/ai.ts`：
- OpenAI API Key 已設定
- 模型：gpt-4.1-2025-04-14
- 自動處理錯誤和重試

### 數據存儲
- 使用瀏覽器 localStorage
- 數據完全本地化
- 無需雲端服務器
