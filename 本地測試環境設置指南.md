# Mind-Brain 本地測試環境設置指南

## 📋 環境需求

- **Node.js**: v16 或更高版本 (推薦 v18+)
- **npm**: v7 或更高版本
- **作業系統**: Windows、macOS 或 Linux

## 🚀 快速啟動

### 方法一：使用一鍵啟動腳本 (推薦)

```bash
# 給腳本執行權限 (macOS/Linux)
chmod +x start-local-test.sh

# 啟動測試環境
./start-local-test.sh
```

### 方法二：手動步驟

1. **安裝依賴**
   ```bash
   npm install
   ```

2. **設置環境變數**
   - 複製 `.env.example` 為 `.env`
   - 編輯 `.env` 文件設置您的 API Keys

3. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

## 🔧 API Key 獲取方式

### OpenAI API Key
1. 前往 [OpenAI Platform](https://platform.openai.com/api-keys)
2. 登入帳號後點選 "Create new secret key"
3. 複製 API Key 並填入 `.env` 文件

### Notion OAuth (可選)
1. 前往 [Notion Integrations](https://www.notion.so/my-integrations)
2. 點選 "New integration"
3. 獲取 Client ID 和 Client Secret

## 🛠️ 常見問題

### Q: 開發伺服器無法啟動
**A:** 
- 檢查 Node.js 版本 (`node --version`)
- 重新安裝依賴 (`rm -rf node_modules && npm install`)
- 檢查端口 3000 是否被佔用

### Q: AI 功能無法使用
**A:** 
- 確認 `.env` 中的 `VITE_OPENAI_API_KEY` 已正確設置
- 檢查 API Key 是否有效且有足夠額度

### Q: 資料無法保存
**A:** 
- 確認網路連線正常
- 檢查 Supabase 服務狀態
